# Restaurant-Order-Analysis-Using-SQL
This project demonstrates SQL-based analysis of a restaurant database using MySQL Workbench. The objective is to extract meaningful business insights from restaurant menu and order data through SQL queries.
# Database Used
## restaurant_db
Tables Analyzed
# 1. menu_items

# Contains information about:

Menu Item Name
Category
Price
# 2. order_details

Contains information about:

Order ID
Order Date
Item ID
Customer Orders
Analysis Performed
Menu Analysis
View all menu items
Count total menu items
Identify least expensive and most expensive menu items
Find the number of dishes available in each category
Analyze Italian menu items
Explore menu pricing structure
Order Analysis
View complete order details
Determine order date range
Calculate total number of orders
Calculate total number of items ordered
Identify orders with the highest number of items
Find orders exceeding a specific item count
Analyze customer ordering patterns
